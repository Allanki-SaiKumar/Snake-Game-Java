# 🐍 Snake Game

A classic Snake Game developed using **Java Swing** and **Object-Oriented Programming (OOP)**. Control the snake, collect food, grow longer, and avoid collisions to achieve the highest score.

## ✨ Features

- 🎮 Smooth and responsive snake movement
- 🍎 Random food generation
- 📈 Real-time score tracking
- 💥 Collision detection
- ⌨️ Keyboard controls
- 🖥️ Interactive graphical interface using Java Swing

## 🛠️ Technologies Used

- Java
- Java Swing
- AWT
- Object-Oriented Programming (OOP)

## 🚀 Getting Started

### Prerequisites
- Java JDK 8 or above
- Any Java IDE (VS Code, IntelliJ IDEA, Eclipse, NetBeans)

### Run the Project

1. Clone the repository:

```bash
git clone https://github.com/Allanki-SaiKumar/Snake-Game-Java.git
```

2. Open the project in your preferred Java IDE.

3. Compile and run `Snake.java`.

4. Enjoy playing the game!

## 🎮 Controls

| Key | Action |
|------|--------|
| ⬆️ Up Arrow | Move Up |
| ⬇️ Down Arrow | Move Down |
| ⬅️ Left Arrow | Move Left |
| ➡️ Right Arrow | Move Right |
